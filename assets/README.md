# Assets Folder

This folder should contain app assets like:

## Required Assets

1. **icon.png** - App icon (1024x1024)
2. **splash.png** - Splash screen (1242x2436 for iOS, 1080x1920 for Android)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024)
4. **favicon.png** - Web favicon (48x48)
5. **notification-icon.png** - Notification icon (96x96)

## Placeholder Note

For demo purposes, you can use simple placeholder images. In production:
- Replace with actual branded assets
- Use high-resolution PNG files
- Follow platform-specific guidelines

## Image Generation

You can generate icons using:
- **Figma** - Design and export
- **Icon Kitchen** - Android adaptive icons
- **App Icon Generator** - Multi-platform icons

## Assets Structure (Optional)

```
assets/
├── images/
│   ├── onboarding/
│   ├── illustrations/
│   └── placeholders/
├── fonts/           (if custom fonts needed)
├── lottie/          (if using Lottie animations)

```
